Toxicity Prediction Using Machine Learning
Overview

This project builds a machine learning model to classify chemical compounds as Toxic or Non-Toxic using molecular descriptor data. The workflow includes EDA, preprocessing, feature selection, class balancing using SMOTE, and training a Random Forest model with cross-validation.

Dataset

Molecular descriptor dataset

Target column: Class

0 → NonToxic

1 → Toxic

Workflow

Data loading and inspection

Exploratory Data Analysis (EDA)

Handling missing values

Data balancing with SMOTE

Feature scaling and selection

Random Forest model training

Cross-validation and evaluation
